## ✨ Características Destacadas

Este proyecto no es solo una página de contacto; es un despliegue de habilidades modernas en desarrollo front-end y arquitectura:

*   📱 **Responsive Design Absoluto**: Se adapta de forma milimétrica a pantallas Desktop, Tablet y Mobile (smartphones pequeños inclusive).
*   🎨 **Estética Premium ("Glassmorphism")**: Uso de fondos esmaltados, gradientes luminiscentes sutiles interactivos y sombras calculadas que transmiten profundidad.
*   ⚡ **Animaciones Fluidas (60fps)**: Transiciones `Hero`, desvanecimientos (`Fade-in`), efectos de latidos y textos dinámicos con `AnimatedTextKit`.
*   🏗️ **Arquitectura Limpia y Modular**: Código totalmente desacoplado y organizado. Cada widget es una porción que respeta la modularidad:
    *   `/lib/src/pages/`: Diseño de la estructura principal.
    *   `/lib/src/widgets/`: Componentes universales y reutilizables.
    *   `/lib/src/themes/ & /lib/src/constants/`: Tokens de diseño y *data*.
*   🕵️‍♂️ **Proyectos Místicos**: Una sección de *"Proyectos Destacados"* que captura intriga y llama a la interacción a través de un Copywriting inmersivo.

## 🛠️ Stack Tecnológico

*   **Framework:** [Flutter SDK](https://flutter.dev/) (Canal Estable)
*   **Lenguaje:** Dart
*   **Fuentes del sistema:** Google Fonts (Outfit, Inter)
*   **Alojamiento (Deploy):** Firebase Hosting / GitHub Pages (Configurable)

## 📸 Estructura de Visualización

1.  **Banner Principal (Hero):** Avatares animados, texto dinámico con los roles (Technical Lead, Senior Flutter Developer, Backend, AWS Cloud) y botones de Call-To-Action (Proyectos & LinkedIn).
2.  **Habilidades (Tech Stack):** Chips visuales que representan las herramientas que domino con iconos intuitivos.
3.  **Experiencia Laboral:** Timeline (Línea del tiempo) elegante enumerando los saltos más destacados de la carrera.
4.  **Prohibido / En Desarrollo:** Una cartelera misteriosa y épica informando de los grandes proyectos que se vienen operando en absoluta confidencialidad. 
5.  **Catálogo de App:** Portafolios Hobby y de grandes dimensiones.

## 🚀 Instalación y Despliegue Local

Para correr este proyecto en tu máquina local:

1.  Asegúrate de tener Flutter instalado y en el [canal estable](https://docs.flutter.dev/release/archive).
2.  Clona el repositorio:
    ```bash
    git clone https://github.com/OdaFra/portfolio.git
    cd portfolio
    ```
3.  Instala las dependencias necesarias de pubspec.yaml:
    ```bash
    flutter pub get
    ```
4.  Lanza el proyecto directamente en Chrome o Edge:
    ```bash
    flutter run -d chrome
    ```
